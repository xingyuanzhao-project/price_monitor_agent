price monitor is the prototype of this project, which is offline and not in production and only applied to very limited sources and not agentic, but has the general idea and plasn for price monitor agents.

nocode-workflow is the project that is webapp that use drag and drop to create workflows, which is a comparable product to this project's final output product.

This project, price monitor agent, is expected to:
1. develop from the idea of price monitor, but need to have full range of modules that does front end and back end, and also have agentic features.
2. should have similar deliverables like nocode-workflow, such as documentations pages, demo on hugging face, and web app in production.

deviation from nocode-workflow:
- it process data unit by unit, and feed in llm processors, which is more academic. This project should process data in more industrial way, and data should be processed in more efficient way, batched and chunked efficiently.

deviation from price monitor:
- it is not agentic, and schema and tools are limited and not flexible. This project is taking the idea of price monitor, but not obliged to follow the same architecture and design.
